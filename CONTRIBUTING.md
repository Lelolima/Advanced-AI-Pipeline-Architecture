# Guia de Contribuição

Obrigado por contribuir com o **Advanced AI Pipeline Architecture**! 🎉

## Como Contribuir

### 1. Fork e Clone

```bash
# Faça fork no GitHub e clone
git clone https://github.com/SEU-USUARIO/Advanced-AI-Pipeline-Architecture.git
cd Advanced-AI-Pipeline-Architecture
```

### 2. Crie uma Branch

```bash
# Crie uma branch para sua feature
git checkout -b feature/minha-feature-incrivel
```

### 3. Desenvolva

- **Frontend (React/TypeScript)**: Siga as convenções do projeto
- **Backend (Python)**: Adicione testes para novas funcionalidades

### 4. Commit

```bash
# Use mensagens descritivas
git commit -m "feat: adiciona análise de sentimentos no módulo LLM"
```

### 5. Push e Pull Request

```bash
git push origin feature/minha-feature-incrivel
# Abra um PR no GitHub
```

## Padrões de Código

### TypeScript (Frontend)

- Use TypeScript estrito (`strict: true`)
- Nomeie componentes em PascalCase
- Use hooks do React de forma consistente
- Adicione tipos explícitos para props e estados

### Python (Backend)

- Siga PEP 8
- Use type hints
- Documente funções com docstrings
- Adicione testes unitários

## Estrutura de Commits

```
feat: Nova funcionalidade
fix: Correção de bug
docs: Mudança na documentação
style: Formatação/cosmético (sem mudança de lógica)
refactor: Refatoração
test: Adição/modificação de testes
chore: Manutenção/build
```
